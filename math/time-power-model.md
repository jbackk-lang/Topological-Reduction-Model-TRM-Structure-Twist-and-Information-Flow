# Model Czasu Potęgowego

Czas w modelu Φ nie jest liniowy.



\[
t_{\text{real}} = t_{\text{obserwowany}}^{\,\alpha}
\]



gdzie α opisuje stopień kompresji czasu przez redukcję informacji.

Wyjaśnia to:
- różnice między czasem geologicznym a topologicznym,
- dlaczego fale przejść zwalniają,
- dlaczego φ jest stanem asymptotycznym.
