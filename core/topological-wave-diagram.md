# Diagram Fali Topologicznej (z przykładami)

Poniższy diagram pokazuje, jak fala topologiczna (zmiana skrętu τ) propaguje się przez układ
i wywołuje przejścia, gdy przekroczy próg krytyczny τ_krit.

Wykres łączy:
- fizykę (rozpady),
- geologię (wymierania),
- biologię (drabina ewolucyjna).

---------------------------------------------------------
                DIAGRAM FALI TOPOLOGICZNEJ
---------------------------------------------------------

        τ
        ↑
        |                         /\
        |                        /  \
        |     A1                /    \      A2
        |    / \               /      \    / \
 τ_krit |---/---\-------------/--------\--/---\---------
        |  /     \           /          \/     \
        | /       \         /                  \
        |/         \       /                    \
        +------------------------------------------------→ t
           t1       t2    t3                    t4

Legenda:
- A1, A2 — amplitudy fali skrętu
- τ_krit — próg przejścia topologicznego
- t1, t2, t3 — momenty przejść
- t4 — fala poniżej progu (brak przejścia)

---------------------------------------------------------
# 1. Interpretacja ogólna

Fala topologiczna to zmiana skrętu:

    τ(x,t) = τ₀ + A cos(kx − ωt)

Przejście zachodzi, gdy:

    |τ(x,t)| > τ_krit

Wtedy układ musi zredukować informację → następuje reset struktury.

---------------------------------------------------------
# 2. Przykłady fizyczne

### (a) Rozpad cząstki
- τ rośnie powyżej τ_krit → cząstka niestabilna
- przejście: τ → μ → e
- przykład: rozpad mionu

### (b) Serie rozpadu pierwiastków
- fala skrętu przekracza próg stabilności jądra
- przejście: U-238 → Th-234 → ... → Pb-206
- każde przejście to redukcja τ

---------------------------------------------------------
# 3. Przykłady geologiczne

### (a) Perm–Trias (252 mln lat temu)
- fala osiąga bardzo wysoką amplitudę A1
- przekracza τ_krit globalnie
- efekt: największe wymieranie w historii Ziemi

### (b) Trias–Jura (201 mln lat temu)
- fala mniejsza, ale nadal powyżej progu
- reset ekosystemów, dominacja dinozaurów

### (c) Kreda–Paleogen (66 mln lat temu)
- fala A2 przekracza próg
- reset: koniec dinozaurów → ssaki

### (d) Obecnie (t4)
- fala poniżej τ_krit
- brak globalnych przejść
- wejście w fazę φ

---------------------------------------------------------
# 4. Przykłady biologiczne

### (a) Dinozaury → Ptaki
- fala skrętu przekracza próg dla dużych form
- redukcja: miniaturyzacja, skrzydła, pióra

### (b) Ptaki → Ssaki
- kolejna fala, mniejsza
- redukcja: stabilizacja metabolizmu, mózg, termika

### (c) Ssaki → Człowiek
- fala nie przekracza progu
- redukcja wewnętrzna (genom, regulacja)
- człowiek = stan bliski φ

---------------------------------------------------------
# 5. Najkrótsze podsumowanie

Fala topologiczna:
- rośnie i maleje jak fala sinusoidalna,
- wywołuje przejścia tylko gdy przekroczy τ_krit,
- każde przejście to redukcja skrętu,
- fale zwalniają w czasie geologicznym,
- w stanie φ fale nie przekraczają progu → brak przejść.
